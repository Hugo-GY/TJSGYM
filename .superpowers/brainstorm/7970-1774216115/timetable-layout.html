Timetable — Layout Approach

How should we organise the weekly schedule? Three approaches:

A

Per-Day Tables (Recommended)

Four separate tables — Monday through Thursday. Each shows Time · Class · Age. Clean, fast to scan by day. Familiar to parents who already know which day they want.

B

Visual Week Grid

Horizontal grid with days as columns and time as rows. Shows the full week at a glance. More visual — good for spotting gaps or comparing days. More complex to build and harder on mobile.

C

Filter + Single Table

One combined table with filter chips by class type (All / Tiddler / Toddler / Mini Gym / Gymnastics). Parents filter to see only their class across all days. Requires JS but very clear for specific-class searches.
