# Markaworks Pupbar Colour Scheme

Source reference: https://markaworks.com/portfolio/pupbar/?utm_source=Pinterest&utm_medium=organic

This palette is an extracted approximation based on the visual material in this folder and the referenced Pupbar page. These hex values are intended as practical working colours rather than exact brand specifications.

## Core Palette

| Role | Approx Hex | Notes |
| --- | --- | --- |
| Deep plum | `#7A2A7A` | Primary logo/text colour, strongest brand anchor |
| Bright magenta | `#D94A9F` | Main high-energy accent, cards/posters/signage |
| Soft blush pink | `#F3C7E7` | Large background fields and gentle highlight areas |
| Light pink wash | `#F7E9F5` | Clean soft surface colour |
| Mint teal | `#39C39A` | Secondary playful accent |
| Acid lime | `#DDEB3C` | High-contrast supporting accent |
| Warm cream | `#F5EFE7` | Neutral background tone behind palettes and layouts |

## Colour Character

- The identity is built around a playful clash between `#7A2A7A` and `#D94A9F`.
- Pink is used as the emotional base: warm, friendly, soft, slightly retro.
- Teal and lime are supporting colours, used sparingly to add freshness and energy.
- Cream and pale pink keep the system approachable instead of overly saturated.

## Suggested Usage Split

- 35% `#F3C7E7` or `#F7E9F5` for backgrounds
- 25% `#7A2A7A` for headings, logo, key shapes, contrast blocks
- 20% `#D94A9F` for CTAs, feature panels, promotional highlights
- 10% `#39C39A` for secondary accents or category coding
- 5% `#DDEB3C` for punchy highlights only
- 5% `#F5EFE7` for neutral breathing space

## Practical TJSGYM Adaptation Direction

If you want to borrow the feel rather than copy the brand directly, the useful takeaway is:

- Keep a soft pink or warm pastel base
- Pair it with one strong purple-plum anchor
- Add one fresh playful contrast colour like teal
- Use one high-voltage accent like lime only in small touches
- Avoid cold greys; use warm off-whites instead
